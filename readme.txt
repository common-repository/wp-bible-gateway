== Description ==

Automatically creates links to http://biblegateway.com from references in wordpress posts. It will use which every version of the bible you last queried in bible gateway.

== Installation ==

1. Download wp-bible-gateway.php and place in wp-content/plugins
2. Log into your WordPress admin panel
3. Go to Plugins and "Activate" the plugin
